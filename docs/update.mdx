---
title: "Update eGovernmentOS"
description: "How to update to the latest version of eGovernmentOS"
---

<Warning>
  **Using eGovernmentOS v137?** Please upgrade to v142 or above. We released a major update with significant improvements in v142. To check your version, go to **Settings > About eGovernmentOS**.
</Warning>

## macOS

eGovernmentOS on MacOS should've updated over-the-air. But you can download the latest version directly:

<CardGroup cols={2}>
  <Card title="Download from eGovernmentOS" icon="globe" href="https://egovernmentos.com">
    Get the latest macOS build
  </Card>
  <Card title="GitHub Releases" icon="github" href="https://github.com/mukira/eGovermentOS/releases/latest">
    Download from GitHub
  </Card>
</CardGroup>

## Windows & Linux

Please download the latest binary and install:

<CardGroup cols={2}>
  <Card title="Download from eGovernmentOS" icon="globe" href="https://egovernmentos.com">
    Get the latest build
  </Card>
  <Card title="GitHub Releases" icon="github" href="https://github.com/mukira/eGovermentOS/releases/latest">
    Download from GitHub
  </Card>
</CardGroup>

<Tip>
  After installing, you'll need to reconfigure your LLM provider settings as they won't carry over from the previous version.
</Tip>
