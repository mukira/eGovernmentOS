## Overview

The Model Context Protocol (MCP) allows eGovernmentOS to securely connect with your existing ministry databases, internal tools, and legacy systems without moving data.

<Note>
This documentation is currently being updated for the new eGovernmentOS architecture.
</Note>

## Capabilities

- **Connect Databases**: Securely query SQL/NoSQL databases on your private network
- **Integrate APIs**: Connect to internal government APIs
- **File System Access**: Allow governed access to specific file repositories
